Entendendo a diferença entre elementos block e inline

Elementos Block

Exemplos: div, section, ul, li, header, p, etc...

  1. Ocupam 100% da largura do elemento pai;
  2. Ocupam a própria linha, sendo posicionados abaixo do elemento anterior;
  3. A altura padrão é definida pelo tamanho do conteúdo;
  4. É possível alterar as propriedades de tamanho e largura;
  5. É possível alterar todas as propriedades de margem.

Elementos inline

Exemplos: a, span, b, i, etc...

  1. Ocupam a largura do próprio conteúdo;
  2. Dois ou mais elementos inline em sequência, são posicionados lado a lado;
  3. Não é possível alterar as propriedades de altura e largura;
  4. Somente é possível alterar as margens horizontais;
  5. Ao aplicar float, automaticamente se transformam em elementos block;
  6. Se comporta como um texto.
